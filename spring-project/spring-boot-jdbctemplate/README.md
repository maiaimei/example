# JdbcTemplate提供的方法

1、execute方法 可以用于执行任何SQL语句，一般用于执行DDL语句。

2、update方法和batchUpdate方法 update方法用于执行新增、修改、删除等语句；batchUpdate方法用于执行批量处理相关语句。

3、query方法及queryForXXX方法 用于执行查询相关语句。

4、call方法 用于执行存储过程、函数相关语句。
